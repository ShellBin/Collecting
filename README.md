# Collecting 2.0 —— 文件、截图收集工具

广泛用于大学电子化作业及截图的收集。具备截图自动压缩，服务器端自动重命名，一键打包下载及上传情况查询等便捷实用功能。

## 项目背景
当前分支为全新开发的 Collecting 2.0 版本，相比较其前身 (旧版本 Collecting)，拥有更全面的功能，更佳的性能和可靠性及更漂亮的 UI，在工程结构上也有长足的进步。
